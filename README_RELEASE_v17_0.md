# Padel Tournament OS — Release v17.0

## What changed
- Home rebuilt around one selected live tournament hero card
- Quick stats reduced to three glow chips: Live now / Open / My matches
- My next match promoted as the second decision block on Home
- Profile cleaned from service/debug action and rebuilt as a player card
- Bottom navigation kept as the main shell: Home / Matches / Tournaments / Profile
- Tournament logic preserved from previous versions

## Product rule
- Admin tools stay off Home
- Home is player-first and action-first
- Visual direction: neon dark mode, glow, glass, depth
