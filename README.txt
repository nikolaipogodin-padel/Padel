Padel Riga — локальная версия

Файлы:
- index.html
- styles.css
- app.js

Как использовать:
1. Открой index.html в браузере.
2. Добавляй игроков вручную или нажми "Демо 20".
3. Нажми "Сформировать турнир".
4. Все данные хранятся локально в браузере.
5. Кнопка "Сбросить" очищает данные.

Логика:
- количество игроков должно быть кратно 4
- пары формируются случайно
- группы формируются автоматически
- результат может вводить только игрок матча
