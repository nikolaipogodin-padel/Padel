Padel Riga — локальная версия

Файлы:
- index.html
- styles.css
- app.js

Как использовать:
1. Открой index.html в браузере.
2. Добавь игроков или нажми "Демо 20".
3. Укажи дату, старт и длительность турнира: 1.5 / 2 / 3 часа.
4. Нажми "Сформировать турнир".
5. Все пары играют со всеми по кругу.
6. Количество кортов = количество игроков / 4 с округлением вниз.
7. Игроки сверх полного набора по 4 человека попадают в очередь по порядку регистрации.
8. Данные сохраняются локально в браузере через localStorage.

Кнопка "Сбросить" очищает сайт до стартового состояния.
