Padel Riga — локальная версия

Файлы:
- index.html
- styles.css
- app.js

Как использовать:
1. Открой index.html в браузере.
2. Все данные сохраняются локально в браузере через localStorage.
3. Кнопка "Сбросить данные" очищает сайт до стартового состояния.

Для публикации:
- просто загрузи эти 3 файла в GitHub Pages
- внешние базы данных не нужны
